# LangChain & LangGraph 에이전트 스터디

이 저장소는 LangChain 및 LangGraph의 문법을 공부하고 에이전트 개발을 위한 다양한 테스트를 수행하기 위한 공간입니다.

## 빠른 시작

이 저장소는 `uv`를 사용하여 파이썬 환경을 관리합니다.

1.  **가상환경 생성 및 활성화**

    프로젝트 루트 디렉토리에서 다음 명령어를 실행하여 가상환경을 생성하고 활성화합니다.

    ```bash
    # 가상환경 생성
    uv venv
    # 가상환경 활성화
    source .venv/bin/activate
    ```

2.  **의존성 동기화**

    `uv sync` 명령어를 사용하여 `pyproject.toml`에 명시된 의존성을 설치합니다.

    ```bash
    uv sync
    ```

## 디렉토리 구조

-   `src/`: 테스트 코드를 포함하며, 주로 주피터 노트북 형식으로 작성됩니다.
-   `src/util/`: 디버깅, 출력, 로깅 등과 같은 커스텀 유틸리티 함수를 포함합니다.
-   `.env`: API 키와 같은 환경 변수를 저장하는 파일입니다. 이 파일은 민감한 정보를 포함하므로 저장소에 포함되지 않습니다.

## 목적

LangChain 및 LangGraph의 핵심 개념을 이해하고 실험하여 다양한 유형의 에이전트를 구축하고 테스트하는 것을 주된 목표로 합니다.
